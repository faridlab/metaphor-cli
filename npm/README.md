# @metaphor/metaphor-cli

npm wrapper for [metaphor-cli](https://github.com/faridlab/metaphor-cli).

On install, this package downloads the matching prebuilt binary from GitHub
releases for your platform and exposes it as the `metaphor` command.

```bash
npm install -g @metaphor/metaphor-cli
metaphor --help
```

Supported platforms: macOS (x64, arm64), Linux (x64, arm64).
